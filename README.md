# RTL Free — إضافة الاتجاه العربي الاحترافية

> إضافة متصفح مجانية تُصلح اتجاه النصوص العربية على **جميع المواقع**
> مع خطوط عربية منتقاة، وكاشف ذكي لخطوط النظام، وتحكّم دقيق في المظهر.
>
> **تصميم: شؤون إسلامية**

---

## المميزات الرئيسية

### ١. التفعيل البسيط (زر واحد)
- الإضافة **مُعطَّلة افتراضيًا** على جميع المواقع
- افتح الموقع → اضغط أيقونة RTL Free الحمراء → اضغط **الزر الأخضر/الأحمر الكبير**
- تفعيل وتعطيل بضغطة واحدة لكل موقع
- لا توجد تبديلات مُربِكة (global vs. per-site)

### ٢. محرك كشف ذكي
- كشف تلقائي للنصوص العربية، العبرية، الفارسية، الأردية، الكردية، البشتو، السريانية
- تطبيق `dir="auto"` على العناصر التي تحتوي على RTL
- دعم كامل لـ Shadow DOM والعناصر الداخلية (مواقع الذكاء الاصطناعي الحديثة، المنتديات، …)
- مراقبة مستمرة للمحتوى الديناميكي عبر `MutationObserver` مع `requestIdleCallback`

### ٣. خطوط عربية منتقاة + كاشف ذكي لخطوط النظام
**الخطوط الجاهزة** (Google Fonts — تُحمَّل عند الطلب):
- **خط الموقع الافتراضي** (بدون تغيير — مفعَّل افتراضيًا)
- **IBM Plex Sans Arabic**
- **Tajawal**
- **Amiri**
- **Vazirmatn**

**كاشف خطوط النظام (ميزة حصرية):**
- استخدام `window.queryLocalFonts()` الحديث (Chrome 103+)
- احتياطي يعتمد على Canvas لفحص ~80 خطًا عربيًا شائعًا
- عرض النتائج كقائمة مع معاينة حيّة لكل خط

### ٤. تحكم كامل في الطباعة
- حجم الخط (60% – 200%)
- سُمك الخط (300 – 800)
- تباعد الأسطر (1.0 – 3.0)
- تباعد الأحرف (−0.05 – 0.20)
- إخفاء/إظهار التشكيل (الحركات)
- تحويل الأرقام (عربية ٠١٢ / لاتينية 0 1 2)

### ٥. ميزات متقدمة
- **CSS مخصص** — محرر كامل للمستخدم المتقدم
- **قائمة سياقية** (نقرة يمين) للوصول السريع
- **تصدير/استيراد الإعدادات** بصيغة JSON
- **شارة الأيقونة** تعرض `ON` للمواقع المُفعَّلة
- **فرض RTL** (اختياري) على الصفحة كاملة
- **حماية الأكواد البرمجية** — تبقى LTR افتراضيًا

## التثبيت (التحميل المحلي)

### Chrome / Edge / Brave / Opera

1. افتح رابط الإضافات:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
2. فعّل **وضع المطوّر** (Developer Mode)
3. اضغط **تحميل غير مُحزّم** (Load unpacked)
4. اختر مجلد الإضافة

### Firefox

1. افتح `about:debugging#/runtime/this-firefox`
2. اضغط **تحميل إضافة مؤقتة**
3. اختر ملف `manifest.json`

## الاستخدام

1. افتح أي موقع يحتوي على نصوص عربية
2. اضغط أيقونة RTL Free الحمراء (R) في الشريط العلوي
3. اضغط **الزرّ الكبير** — سيتحوّل للأخضر وتصبح الإضافة فعّالة على هذا الموقع
4. عدِّل الخط والحجم من قائمة الإعدادات

## آلية العمل

### عند تحميل الصفحة
- يُحقن `content.js` مبكرًا (`document_start`)
- يُحمَّل إعداد `enabledSites` من `chrome.storage.sync`
- إذا لم يكن الموقع الحالي ضمن القائمة → لا يحدث شيء

### عند التفعيل
- `dir="auto"` يُطبَّق على كل عنصر يحتوي نصًا عربيًا
- `unicode-bidi: plaintext` للسلامة
- إن اختار المستخدم خطًا → يُحقن `<link>` لـ Google Fonts
- إن كان "خط الموقع (بدون تغيير)" → لا يُلمس خط الموقع

### المحتوى الديناميكي
- `MutationObserver` يراقب التعديلات
- `requestIdleCallback` للتنفيذ دون حجب الصفحة
- يمرُّ على كل `shadowRoot` تلقائيًا

## الخصوصية

- **لا تجمع الإضافة أي بيانات شخصية.**
- كل الإعدادات تُحفظ محليًا عبر `chrome.storage.sync`
- الاتصال الخارجي الوحيد هو Google Fonts (اختياري، عند اختيار خط)
- الكود مفتوح ومقروء للمراجعة
